Films Editor

Edit your films here, then click Download films.json and replace the films.json file in your site folder (then re-deploy). This page auto-loads the current films.json if you open it through the local server; otherwise click Load films.json to pick the file.

Custom thumbnail: drop an image into assets/thumbs/ and put its path in the Thumbnail box, e.g. assets/thumbs/my-film.jpg. Leave it blank to use the automatic YouTube thumbnail.
